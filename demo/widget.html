OmniChat Visitor Widget

Welcome to the OmniChat Visitor Widget Demo! Look at the bottom-right corner of your screen to interact with the chat bubble.

This page demonstrates how easily you can embed the compiled Vue 3 Custom Web Component into any static HTML site using our jsDelivr CDN link.

Setup Instructions:

  1. Include the <script> tag pointing to the CDN in your <head>.
  2. Add the <omnichat-widget> element anywhere in your body.
  3. Point the server-url attribute to your running OmniChat API backend (e.g., http://localhost:3001).
