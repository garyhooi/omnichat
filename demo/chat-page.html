OmniChat Chat Page

The Chat Page opens a full-page live chat in a new tab — perfect for embedding links on your website that visitors can open to start a conversation.

It's also available as an embeddable custom element: <omnichat-chat-page>.

Open Chat in New Tab

Click the button below to open the full-page chat in a new tab.

Open Live Chat

Embed as Custom Element

Add the component directly to any page:

<script src="omnichat-chat-page.js"></script>

<omnichat-chat-page
  server-url="http://localhost:3001"
  bubble-color="#4F46E5"
  welcome-message="Hi there! How can we help?">
</omnichat-chat-page>
